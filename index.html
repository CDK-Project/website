Welcome to CDK

The C/C++ Development Kit (CDK) is a simple cross-platform development environment for C++ developers. It includes a compiler,core libraries, and a package manager, all bundled together.

Features:

  • Similiar to .NET for C#
  • Easy installation and setup
  • Comes with GCC, GDB, vcpkg package manager, and core libraries for development
  • Supports cross-platform development on Windows, Linux, and macOS
